# Flappy Cat

A Flappy Bird clone featuring a flying superhero cat with a rainbow background.

## Features

- Animated flying cat with cape and wings
- Rainbow gradient background with sparkle effects
- Pipe obstacles with gap navigation
- Score tracking and high score
- Start screen and game over screen

## Tech Stack

- Python 3
- Pygame

## Getting Started

### Prerequisites

- Python 3.10+
- Pygame

### Installation

```bash
pip install pygame
```

### Run the Game

```bash
python flappy_cat.py
```

## Controls

- **Space** or **Click** - Flap to fly upward
- Navigate through pipe gaps to score points
- Avoid hitting pipes or screen boundaries

## Game Settings

| Setting | Value |
|---------|-------|
| Screen Size | 400 x 600 |
| Pipe Gap | 180 pixels |
| Pipe Speed | 3 |
| Gravity | 0.5 |
| Flap Strength | -9 |
